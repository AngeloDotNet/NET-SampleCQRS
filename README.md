# Sample API CQRS implementation in C#

A lightweight implementation of the Command Query Responsibility Segregation (CQRS) pattern in C#. This sample project demonstrates how to separate read and write operations in an application using the CQRS pattern, along with technologies such as ASP.NET Core, Entity Framework Core, MediatR, FluentValidation, and RabbitMQ.

## 🏷️ Introduction

This is a sample implementation of the Command Query Responsibility Segregation (CQRS) pattern in C#. The CQRS pattern is a design pattern that separates the read and write operations of an application into different models. This allows for better scalability and performance, as well as improved maintainability.

## 🛠️ Technologies Used

- .NET 8
- ASP.NET Core
- Entity Framework Core
- MediatR
- FluentValidation
- RabbitMQ
